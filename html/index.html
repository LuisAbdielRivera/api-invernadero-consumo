Bienvenido al Invernadero

Un invernadero es una estructura cerrada que permite controlar el clima y otros factores ambientales para cultivar plantas en condiciones óptimas. ¡Explora nuestras secciones para obtener más información sobre el control, reportes y usuarios del invernadero!

Puedes comenzar visitando la página de Control para gestionar los parámetros ambientales, o dirigirte a la sección de Reportes para revisar el historial de datos del invernadero.
